# Erlang

**Manager:** kerl | **Default:** 28.1 | **Location:** `/.sprite/languages/erlang/kerl`

## Version Management

```bash
kerl-helper list            # List installed/available
kerl-helper install 26.2    # Install
kerl-helper activate 26.2   # Switch
kerl-helper update          # Update releases list
```
